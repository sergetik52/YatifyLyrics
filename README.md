# Yatify Discord Activity

Yatify показывает музыку, которую ты слушаешь сейчас: в Discord Activity и/или в Telegram-канале.

Программа берёт текущий трек из Windows Media Session, ищет синхронизированный текст песни, обложку, обновляет Discord Activity, а в Telegram может делать live-канал с названием трека, обложкой, прогресс-баром и строками песни.

## Что умеет

- Показывать текущий трек и строку песни в Discord Activity.
- Работать с Яндекс Музыкой, Spotify, Apple Music, VK Music, YouTube Music и другими плеерами, которые видны Windows.
- Искать и кешировать тексты песен.
- Искать и кешировать обложки.
- Обновлять название и аватарку Telegram-канала.
- Создавать live-сообщение-плеер в Telegram с прогрессом трека.
- Писать строки песни в Telegram-канал.
- Прикреплять Telegram-канал к профилю, если используется режим аккаунта.
- Запускаться скрыто вместе с Windows.

## Быстрая установка

Скачай релизный архив, распакуй его и запусти:

```bat
dist\YatifySetup.exe
```

Мастер на русском спросит, что нужно включить:

- только Discord Activity;
- только Telegram-канал;
- Discord и Telegram вместе.

Потом он создаст `config.json`, предложит войти в Telegram, добавит программу в автозапуск и запустит её скрыто.

Настройки можно поменять в любой момент: просто запусти `dist\YatifySetup.exe` ещё раз.

## Нужно ли что-то устанавливать

Если ты скачал готовый релизный zip, то Python и библиотеки ставить не нужно. Всё уже упаковано в `.exe`.

Нужно только:

- Windows 10/11;
- установленный и запущенный Discord, если нужен Discord Activity;
- включённая активность в настройках Discord;
- музыкальное приложение, которое видно Windows, например Яндекс Музыка или Spotify;
- Telegram-данные, если нужен Telegram-канал.

## Telegram

Telegram необязателен. Можно пользоваться только Discord.

Есть два режима Telegram.

### Режим аккаунта

Этот режим использует твой Telegram-аккаунт через официальный MTProto API. Он нужен, если ты хочешь, чтобы канал прикреплялся к профилю.

Что нужно:

1. Зайти на `https://my.telegram.org`.
2. Получить `api_id` и `api_hash`.
3. Запустить `dist\YatifySetup.exe`.
4. Выбрать Telegram через аккаунт.
5. Ввести `api_id`, `api_hash`, телефон и пройти вход.

Аккаунт должен быть владельцем или админом канала.

### Режим бота

Режим бота проще, но бот не может прикреплять канал к профилю.

Что нужно:

1. Создать бота через `@BotFather`.
2. Добавить бота в Telegram-канал админом.
3. Дать права на публикацию, редактирование, удаление сообщений и изменение канала.
4. Запустить `dist\YatifySetup.exe`.
5. Ввести token бота и канал.

## Что публикуется в Telegram

Если Telegram включён, программа может:

- менять название канала на `Артист - Трек`;
- менять аватарку канала на обложку трека;
- удалять служебные сообщения Telegram про смену названия, аватарки и закрепа;
- закреплять live-плеер с прогрессом трека;
- отправлять строки песни отдельными сообщениями;
- чистить старые строки при смене трека.

Telegram может давать лимиты за слишком частые действия. Поэтому по умолчанию плеер обновляется раз в `15` секунд.

## Ручная настройка

Можно не пользоваться мастером, а скопировать:

```text
config.example.json
```

в:

```text
config.json
```

и заполнить руками.

Пример:

```json
{
  "discord_enabled": true,
  "telegram_mode": "off",
  "telegram_api_id": "",
  "telegram_api_hash": "",
  "telegram_phone": "+79990000000",
  "telegram_channel": "@your_channel_username",
  "telegram_personal_channel": true,
  "telegram_proxy": "",
  "lyrics_use_proxy": true,
  "telegram_player_update_interval": 15.0,
  "telegram_player_bar_width": 10,
  "cache_miss_ttl": 86400.0,
  "telegram_bot_token": "",
  "telegram_chat_id": "@your_channel_username"
}
```

Важно: никогда не выкладывай `config.json` в интернет. Там могут быть Telegram token, `api_hash`, телефон и другие приватные данные. Файл уже добавлен в `.gitignore`.

## Сборка из исходников

Если хочешь собрать `.exe` сам, установи Python 3.12+ или 3.13 и запусти:

```bat
build_exe.bat
```

После сборки появятся:

- `dist\YaMusicLyricsDiscordActivity.exe` - основная программа;
- `dist\TelegramLogin.exe` - вход в Telegram-аккаунт;
- `dist\YatifySetup.exe` - мастер настройки.

Если pip работает только через прокси, можно указать его перед сборкой:

```powershell
$env:PIP_PROXY = "http://user:pass@host:port"
.\build_exe.ps1
```

## Создание релизного архива

После сборки запусти:

```bat
pack_for_friend.bat
```

Архив появится рядом с папкой проекта.

В релизный архив попадают только безопасные файлы и готовые `.exe`. Туда не попадают:

- `config.json`;
- логи;
- кеш;
- Telegram-сессии;
- `.venv`;
- `.buildvenv`;
- `build`;
- локальные временные файлы.

## Архив исходников для GitHub

Чтобы собрать чистый архив исходников:

```bat
pack_source.bat
```

Он создаст `Yatify-DiscordActivity-source.zip` без приватных данных и без собранных файлов.

## Запуск и остановка

Добавить в автозапуск и запустить:

```bat
install_autostart.bat
```

Остановить и убрать из автозапуска:

```bat
uninstall_autostart.bat
```

Запустить вручную:

```bat
start_hidden.bat
```

## Логи и кеш

Логи и локальная база лежат тут:

```text
logs\app.log
logs\cache\
logs\telegram_user.session
```

Эти файлы нужны только на твоём ПК и не должны попадать в репозиторий.

## Важные замечания

- Программа не использует Discord user token.
- Discord должен быть запущен.
- В Discord должна быть включена активность.
- Если текст или обложка уже найдены один раз, они сохраняются локально и повторно не ищутся в интернете.
- Если Telegram дал flood limit, лучше временно остановить программу и подождать.
